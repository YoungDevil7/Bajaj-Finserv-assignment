# BFHL Client (Frontend)

This is the frontend for the SRM Full Stack Engineering Challenge.

## How to run

```
cd client
npm install
npm start
```

- The frontend will be available at http://localhost:3000 or the port you choose.
- Make sure the backend server is running at http://localhost:3001 for API calls.

## Features
- Input field for node edges (one per line or comma-separated)
- Submit button to call the /bfhl API
- Displays the API response in a readable format
- Shows error messages on API failure
