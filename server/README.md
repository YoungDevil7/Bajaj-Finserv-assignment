# BFHL API Server

This is the backend for the SRM Full Stack Engineering Challenge.

## How to run

```
cd server
npm install
npm start
```

- The API will be available at http://localhost:3001/bfhl
- CORS is enabled for frontend integration.

## Endpoint
- POST /bfhl
- Content-Type: application/json
